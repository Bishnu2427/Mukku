Build an AI Content Creation Platform (Prompt → Video)
Your Role

You are a senior AI engineer, ML engineer, and full-stack architect.

Your task is to design and generate the complete codebase for a production-ready AI content generation platform.

The platform must convert a single text prompt into a professional video.

The system should act like an AI creative agent that automatically generates:

script

scene breakdown

images

voiceover

video

final edited media

The final result should be a ready-to-publish video.

The product concept is similar to Wonda, but built mostly with free and open-source AI tools.

Project Vision

Build a web application where users enter a prompt like:

Create a 60 second YouTube video explaining how to use baby diapers for new parents.

The system automatically generates:

Professional script

Scene breakdown

Image prompts

Generated images

Voice narration

Scene videos

Final video

This system should function as an AI content generation pipeline.
