AI Content Creation Platform (Prompt → Video Generator)
Role

You are a senior AI engineer and full-stack architect.

Your task is to design and implement a production-quality AI content creation web application that generates professional video content from a single prompt.

The system should behave like an AI creative agent that automatically generates:

Script

Scene plan

Images

Voiceover

Video

Final edited output

The final output should be a ready-to-publish video.

The system concept is similar to Wonda, but implemented primarily with free and open-source AI models.

Project Objective

Build a web application where a user enters a prompt describing the content they want, and the system automatically generates a complete video.

Example prompt:

Create a 60 second YouTube video explaining how to use baby diapers for new parents.

The system must automatically perform the following pipeline:

Understand the prompt

Generate a professional script

Break the script into scenes

Generate images for each scene

Generate voiceover narration

Assemble a video

Provide a downloadable final video

AI Model Strategy

The system should prioritize free or open-source models.

LLM for Script Generation

Use a local LLM via Ollama.

Possible models:

Llama3

Mistral

Mixtral

Purpose:

Convert user prompts into professional video scripts.

Scene Planning

Use the LLM to convert scripts into structured scenes.

Example output:

Scene 1
Visual: Mother preparing diaper changing table
Voiceover: Preparing a clean diaper changing station is important.

Scene 2
Visual: Baby lying on changing mat
Voiceover: Place your baby gently on the mat.

Each scene should include:

visual prompt

narration

duration

Image Generation

Generate visuals using Stable Diffusion from Hugging Face.

Example prompt:

mother preparing baby diaper changing station, soft lighting, realistic photography

Output:

scene1.png
scene2.png
scene3.png
Voice Generation

Generate narration using open-source TTS.

Recommended tools:

Coqui TTS

Bark

Output example:

scene1_voice.wav
scene2_voice.wav
Video Assembly

Combine images and audio into a final video.

Use:

MoviePy

FFmpeg

Add:

scene transitions

zoom/pan effects

optional background music

Output:

final_video.mp4

System Behavior Requirements

The system must process requests asynchronously since video generation takes time.

Each pipeline step should save intermediate results.

The system should allow debugging of each step.

Generated assets should be stored locally in structured folders.
